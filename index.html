Portfólio

Churrascômetro

O Churrascômetro é uma aplicação simples desenvolvida para ajudar na organização de churrascos. Com base no número de adultos, crianças e na duração do evento, a aplicação calcula as quantidades necessárias de carne, cerveja e bebidas não alcoólicas. É uma ferramenta prática para quem deseja planejar eventos de forma eficiente, evitando desperdícios ou falta de suprimentos.

Jogo da Memória

Um jogo da memória interativo desenvolvido com HTML, CSS e JavaScript, que utiliza ícones de tecnologias como React, Node.js e Bootstrap para treinar a memória de forma divertida. As cartas são embaralhadas dinamicamente a cada partida, com animações de flip, detecção automática de pares e mensagem de parabéns ao final. Interface responsiva e visual moderno, ideal para praticar lógica de programação e manipulação do DOM.
